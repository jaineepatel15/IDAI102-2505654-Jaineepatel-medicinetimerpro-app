# IDAI102-2505654-Jaineepatel-medicinetimerpro-app
MedTimer Pro is a simple and friendly app that helps users track their daily medicines, mark doses as taken, and stay motivated with progress tracking and streaks, promoting healthy and consistent habits.

Overview:
MedTimer Pro is a simple and easy medicine tracker app made with Python and Streamlit. It helps users remember to take their medicines on time every day. The app shows the list of medicines for the day, allows marking them as taken, and tracks your progress and streak over time.

Purpose and Problem:
Many people forget to take their medicines on time, which can affect their health. The purpose of MedTimer Pro is to help users remember and stay consistent with their medication routine. The app solves this problem by displaying scheduled medicines, allowing users to mark them as taken, and giving motivational progress feedback.

App Features:
Add medicines with name, type (pill, syrup, etc.), time, and dosage
See today’s medicine schedule with clear status (taken, missed, or upcoming)
Mark medicines as taken
Track progress with a percentage and progress bar
View past history of taken medicines
Reset for a new day while keeping streak data
Shows motivational messages and streak count
Option to clear all data

App Flow:
User enters medicine details like name, type, time, and dosage.
The medicine gets added to the list for that day.
The app compares current time with scheduled time and shows medicine status as taken, missed, or upcoming.
User can click a button to mark it as taken.
The app updates the taken status and tracks progress for the day.
After the day ends, the user can click “Start New Day” to reset the list but keep streak info.
The user can view medication history and track consistency over time.

Design Planning
The app uses a calm color theme with a gradient background.
Simple layout with titles, buttons, and progress indicators.
Uses icons to represent medicine types (e.g. pill or syrup).
Motivational messages help encourage users.
Designed to be simple, friendly, and easy to use for all age groups.
